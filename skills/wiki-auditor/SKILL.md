# Wiki Auditor

## Workflow
1. Read trust metadata
2. Review stale pages
3. Review contradictions
4. Check summary coverage
5. Output a prioritized audit report

## Rules
- Review provenance and freshness signals before recommending page changes
- Treat open contradictions and missing summary layers as first-class audit findings
- Prioritize issues that block trust in canonical wiki pages
