# Wiki Distillation

## Workflow
1. Review durable answers or candidate updates
2. Choose the target page type
3. Update or create canonical wiki pages
4. Update summaries and provenance links
5. Record the promotion outcome

## Rules
- Promote only when the evidence is durable enough for canonical wiki content
- Use candidate updates when confidence is not yet high enough for immediate promotion
- Keep summaries and provenance links aligned with the promoted page state
