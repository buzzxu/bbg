# Hermes Outcome Evaluation

## Workflow

1. Select adoption records and measurement windows.
2. Compare baseline and observed metrics for relevant goals.
3. Assign outcome verdicts with confidence and evidence references.
4. Flag regressions for rollback or strategy refinement review.
5. Publish outcome summaries for future K11 learning passes.

## Rules

- outcome verdicts must reference measurable evidence windows
- Regressions require explicit review rather than automatic rollback
- K12 outcome analysis remains evidence-gated and human-reviewed
