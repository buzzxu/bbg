# Wiki Provenance

## Workflow
1. Compare source changes
2. Update freshness status
3. Trace evidence from wiki page to sources
4. Flag stale pages and unresolved evidence gaps

## Rules
- Do not mark a page current when the evidence trail is incomplete
- Record freshness and evidence gaps explicitly instead of hiding uncertainty
- Prefer traceable source paths and hashes over opaque trust judgments
