# Hermes Verification

## Workflow

1. Select intake runs or candidates ready for verification review.
2. Validate evidence quality and source traceability for each candidate.
3. Score correctness, reproducibility, reuse potential, and confidence.
4. Record verification outcomes with reviewer notes and timestamps.
5. Route only verified candidates into promotion decision workflows.

## Rules

- verification must be evidence-gated
- Record correctness, reproducibility, reuse potential, and confidence
- Verification output must be reviewable and auditable
