Implement the requested feature using TDD:

1. **Understand**: Read relevant source files to understand the current codebase
2. **Write Test (RED)**: Create a failing test in `tests/unit/` or `tests/integration/`
3. **Implement (GREEN)**: Write minimal code to make the test pass
4. **Refactor (IMPROVE)**: Clean up while keeping tests green
5. **Verify**: Run `npm test` and `npm run build`

Target 80%+ test coverage for new code.
Follow the patterns in existing test files.
Use vitest as the test framework.
