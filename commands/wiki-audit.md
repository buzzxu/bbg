# /wiki-audit

## Description
Audit the knowledge layer for trust and consistency issues so stale pages, contradictions, missing summary layers, and pending candidate updates can be reviewed systematically.

## Process
1. Review source freshness
2. Review contradictions
3. Review summary-layer completeness
4. Review pending candidate updates
5. Produce a prioritized audit report
