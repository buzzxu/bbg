# /wiki-promote

## Description
Promote a durable answer or candidate update into a canonical wiki change when the evidence is strong enough to justify a maintained page update.

## Process
1. Review the durable answer or candidate update
2. Choose the canonical target page or page type
3. Update or create the formal wiki page
4. Record the promotion outcome and follow-up maintenance notes
