description: Create implementation plan for complex features and refactoring
---

Create an implementation plan for the requested feature.

Follow the workflow in `agents/planner.md` and `commands/plan.md`:

1. Read relevant source files and understand the current architecture
2. Identify which modules and files need to change
3. Check `rules/common/patterns.md` for established patterns
4. Break into phases of 2-4 small, testable tasks
5. Assess risks and edge cases
6. Define what tests need to be written
7. Reference applicable skills from `skills/` for implementation guidance

Output as a numbered task list with files, dependencies, and complexity.
