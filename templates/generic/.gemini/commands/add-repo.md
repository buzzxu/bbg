# Gemini Add Repo Adapter

<!-- BBG:BEGIN MANAGED -->
Run the project-local BBG Add Repo Skill adapter: `.gemini/skills/bbg-add-repo/SKILL.md`.

Use this project-local command when the user asks to add/register a repository, local child project, or remote Git repository.

1. Read `.gemini/skills/bbg-add-repo/SKILL.md`
2. Read `.bbg/harness/skills/add-repo/SKILL.md`
3. Read `AGENTS.md`
4. Read `RULES.md`
5. Infer repository evidence from the current workspace
6. Use `bbg add-repo-agent` only as the skill's internal runner
7. Continue into `.bbg/harness/skills/analyze/SKILL.md` after registration
8. Summarize the added repo, inferred type/stack, analysis status, and unresolved blockers
