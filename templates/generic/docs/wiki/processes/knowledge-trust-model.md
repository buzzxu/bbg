# Knowledge Trust Model

## Freshness

Track source and page `freshness_status` so readers can distinguish current, stale, and review-needed knowledge.

## Contradictions

Record contradictions explicitly instead of hiding them in page prose.

## Summary Layers

Use summary layers to keep L0, L1, and L2 style summaries aligned with canonical pages.

## Promotion Decisions

Promote candidate updates into canonical pages only when the evidence and confidence are strong enough.
